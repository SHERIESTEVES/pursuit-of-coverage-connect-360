# CalAIM Connect360 Platform

CalAIM Connect360 is a practice management, EHR, case management, billing, vendor, and care coordination platform concept designed for California CalAIM Enhanced Care Management, Community Supports, Housing Interventions, Justice-Involved services, and fee-for-service billing workflows.

## Purpose

This platform is designed to support whole-person care coordination, electronic claim billing, documentation compliance, vendor management, member engagement, and DHCS/MCP reporting in one connected system.

## Core Features

- Member / patient portal
- Vendor portal
- Care team collaboration hub
- ECM and Community Supports case management
- Housing Intervention workflows
- Justice-Involved pre-release and post-release coordination
- FFS and MCP billing workflows
- Electronic claims submission
- EDI support
- 837 claim creation
- 835 remittance posting
- 270/271 eligibility verification
- Authorization tracking
- Appointment scheduling
- Escort requests
- Check requests
- Vendor invoicing
- Electronic signatures
- Required documentation templates
- EHR education classes
- Capacity tracking
- DHCS reporting dashboards
- Denial tracking, corrections, and appeals
- Audit-ready documentation trails
